# 🍔 Foodie Express

A modern and responsive food ordering landing page built with **HTML5, CSS3, and JavaScript**. This project provides a clean user interface with food search, shopping cart functionality, smooth scrolling, and dark mode support.

## ✨ Features

- 🍔 Beautiful Hero Section
- 🔍 Search Food Items
- 🛒 Add to Cart Functionality
- 💰 Automatic Total Price Calculation
- 🌙 Dark / Light Mode Toggle
- 📜 Smooth Scroll Navigation
- 📱 Responsive Design
- 🎨 Modern Card Hover Effects
- 🖼️ Image Zoom Animation
- ⚡ Fast and Lightweight
